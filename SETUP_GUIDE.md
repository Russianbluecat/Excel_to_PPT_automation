# 🛠️ 설치 및 설정 가이드

## 📥 설치 방법

### 방법 1: VBA 파일 가져오기 (권장)

1. **Excel 파일 열기**
   - 매크로가 실행될 Excel 파일 열기

2. **VBA 편집기 열기**
   - `Alt + F11` 누르기
   - 또는 `개발 도구` → `Visual Basic` 클릭

3. **모듈 가져오기**
   - VBA 편집기에서 `파일` → `가져오기` 선택
   - `Excel_to_PPT_Automation.bas` 파일 선택
   - `열기` 클릭

4. **확인**
   - 왼쪽 프로젝트 탐색기에서 모듈이 추가되었는지 확인

### 방법 2: 코드 복사-붙여넣기

1. **새 모듈 만들기**
   - VBA 편집기에서 `삽입` → `모듈` 선택

2. **코드 복사**
   - `Excel_to_PPT_Automation.bas` 파일을 텍스트 편집기로 열기
   - 전체 코드 복사 (`Ctrl + A`, `Ctrl + C`)

3. **코드 붙여넣기**
   - VBA 편집기의 모듈에 붙여넣기 (`Ctrl + V`)

4. **저장**
   - `Ctrl + S` 또는 `파일` → `저장`

## ⚙️ 초기 설정

### 1. Excel 시트 준비

```
1. 새 시트 만들기 또는 이름 변경
2. 시트 이름을 원하는 이름으로 지정 (예시 파일에는 Copy_to_Slide 라 되어 있음)
3. 데이터 입력
```

**시트 이름 변경 방법:**
- 시트 탭 우클릭 → `이름 바꾸기`
- 또는 시트 탭 더블클릭

### 2. PowerPoint 객체 이름 설정

**단계별 가이드:**

1. **PowerPoint 파일 열기**

2. **선택창 열기**
   - `홈` 탭 → `편집` 그룹 → `선택` → `선택창`
   - 또는 `Alt + F10`

3. **객체 선택 및 이름 변경**
   ```
   텍스트박스 선택 → 선택창에서 이름 클릭 → "Slide_1_Text_01" 입력
   차트 선택 → 선택창에서 이름 클릭 → "Slide_1_Pie_01" 입력
   ```

4. **명명 규칙**
   ```
   Slide_[슬라이드번호]_[타입]_[순번]
   
   예시:
   - Slide_1_Text_01    (1번 슬라이드, 텍스트박스, 첫 번째)
   - Slide_1_Pie_01     (1번 슬라이드, 파이차트, 첫 번째)
   - Slide_2_Bar_01     (2번 슬라이드, 막대차트, 첫 번째)
   - Slide_2_Line_01    (2번 슬라이드, 선차트, 첫 번째)
   ```

### 3. 파일 경로 설정

**코드에서 수정할 부분:**

```vba
' Update_All_Slides() 함수 내부
pptPath = "여기에_실제_경로_입력.pptx"
```

**파일 경로 얻는 방법:**
1. PowerPoint 파일 우클릭
2. `속성` 선택
3. `위치` 복사 + 파일명 추가

**예시:**
```vba
pptPath = "C:\Users\YourName\Documents\Report\Monthly_Report.pptx"
```

## 🔍 테스트 실행

### 1. 매크로 보안 설정

**Excel 설정:**
1. `파일` → `옵션` → `보안 센터`
2. `보안 센터 설정` 클릭
3. `매크로 설정` → `경고와 함께 모든 매크로 포함` 선택
4. `확인`

### 2. 테스트 실행

1. **Excel에서 매크로 실행**
   - `Alt + F8` 누르기
   - `Update_All_Slides` 선택
   - `실행` 클릭

2. **결과 확인**
   - PowerPoint가 자동으로 열림
   - 데이터가 복사되는지 확인
   - "모든 슬라이드 업데이트 완료!" 메시지 확인

3. **디버그 로그 확인**
   - VBA 편집기에서 `Ctrl + G` (즉시 창 열기)
   - 각 단계별 완료 메시지 확인

## ⚠️ 문제 해결

### "Copy_to_Slide 시트를 찾을 수 없습니다"

**해결 방법:**
- Excel 시트 이름이 정확히 `Copy_to_Slide`인지 확인
- 띄어쓰기, 대소문자 주의
- 또는 코드에서 시트 이름 수정:
  ```vba
  Set ws = ThisWorkbook.Worksheets("실제_시트_이름")
  ```

### "PPT 파일을 찾을 수 없습니다"

**해결 방법:**
- 파일 경로를 복사-붙여넣기로 입력
- 백슬래시(`\`) 확인
- 파일이 실제로 존재하는지 확인

### "Slide_X_XXX_XX를 찾을 수 없음"

**해결 방법:**
- PowerPoint 선택창에서 객체 이름 확인
- 대소문자, 언더스코어(`_`) 정확히 일치하는지 확인
- 객체가 실제로 해당 슬라이드에 있는지 확인

### 차트 데이터가 업데이트되지 않음

**해결 방법:**
- Excel 데이터 범위가 정확한지 확인
- 데이터 형식 확인 (텍스트 vs 숫자)
- 차트 데이터 시트가 열리는지 수동 확인

## 🎓 추가 팁

### 단축키 만들기

1. VBA 편집기에서 코드 상단에 추가:
   ```vba
   Sub Update_All_Slides()
       Application.OnKey "^+U", "Update_All_Slides"  ' Ctrl+Shift+U
       ' ... 기존 코드
   ```

2. 이제 `Ctrl + Shift + U`로 실행 가능

### 자동 저장 경로 설정

```vba
' 파일을 다른 이름으로 저장
pptPres.SaveAs "C:\Backup\Report_" & Format(Date, "yyyymmdd") & ".pptx"
```

### 실행 시간 측정

```vba
Sub Update_All_Slides()
    Dim startTime As Double
    startTime = Timer
    
    ' ... 기존 코드 ...
    
    Debug.Print "실행 시간: " & Round(Timer - startTime, 2) & "초"
End Sub
```
